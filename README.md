# LeetCode Solutions
Hello! Welcome to this repository! This repo houses my proposed solutions for various LeetCode problems as I learn how to code in different programming languages!
I have segregated each solution into ***different programming languages*** if you are interested to see how each solution is presented in different languages.

## How to Use
LeetCode problems can be accessed under the ***Problems*** section [here](https://leetcode.com/problemset/).
1. Scroll down until you see the list of incrementally numbered problems below. Select whichever one you would like to access.
<img width="1918" height="940" alt="image" src="https://github.com/user-attachments/assets/8e18b7cf-4127-47f6-b9af-9a41128ca0f9" />

2. Once you selected a problem, you can see:
   - On the left: Problem **description**, official **editorial** from LeetCode themselves, **solutions** from other programmers on LeetCode, and your past **submissions**.
   - On the top right: The **code** that you will write for your solution to the problem.
   - On the bottom right: The **testcases** you will be testing your code on, which you can add and customize as required, and the **test result** of your written code on the testcases.
   - On the top bar: From left to right are **debug mode** (only available for premium users), **run** your code on the testcases, and **submit** your code as a solution to the problem. 
<img width="1913" height="937" alt="image" src="https://github.com/user-attachments/assets/bde5b4f1-96f6-4b2e-8c0a-fd4df1aa2fd2" />

4. Underneath the code header, you can select **your preferred programming language**! This is where my repository becomes helpful as I continuously update this repository with my proposed solutions. 
<img width="1914" height="938" alt="image" src="https://github.com/user-attachments/assets/e8a53b32-4db8-4c3d-b7e1-65cdde29f90d" />

5. Refer to this repository for my proposed solutions. I have carefully grouped problems by **programming languages** to showcase my proposed solutions in different languages and for ease of navigation!
